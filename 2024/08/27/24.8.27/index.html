搭建仓库

成功下载工具之后，开始搭建 GitHub 存储静态页面的仓库。

  1. 注册或登录 GitHub
  2. 点击 Create a new repository 进入新建仓库页面
  3. 仓库名输入：用户名.github.io
  4. 勾选 Public
  5. 勾选 Add a README file
  6. 拉到页面底部点击 Create 创建仓库
