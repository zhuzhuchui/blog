派生 derived 与继承 inherit

继承是代码复用的一种形式。

继承别人的类叫派生类（derived class 子类），被继承的类叫基类（base class 父类）。

继承以后，派生类和基类就没有关系了，但是派生类可以得到父类的成员变量和成员函数。

继承有什么用呢？
1 、代码复用（省的再写一遍）:
我们看到派生类通过继承就拥有了一些成员变量，成员函数。这都是代码的复用，不必自己重新写一遍。
2 、多态（让程序可扩展）:
多态可以让差异化得到合理的解决，代码更容易读，也更容易写。参考 多态

多继承:
多继承即一个子类可以有多个父类，它继承了多个父类的特性。

C++ 类可以从多个类继承成员，语法如下：

class <派生类名>:<继承方式1><基类名1>,<继承方式2><基类名2>,…
{
<派生类类体>
};

继承方式为权限：

在 C++ 中，有三种访问控制符：

public（公有）：用 public 声明的成员可以从类外部访问。在你的例子中，Base 类中的 int a; 和 void fun(void) 都是 public 成员，因此它们可以在 main() 函数中被访问。

protected（保护）：用 protected 声明的成员只能在类内部或派生类中访问，不能在类外部访问。

private（私有）：用 private 声明的成员只能在类内部访问，不能在类外部或派生类中访问。
